# Animal Sound Files

This directory should contain MP3 files for each animal sound used in the Science game.

The following MP3 files need to be added:

1. dog.mp3
2. cat.mp3
3. cow.mp3
4. duck.mp3
5. frog.mp3
6. lion.mp3

13. bird.mp3

16. wolf.mp3

## Sound Requirements

- Format: MP3
- Duration: 1-3 seconds recommended
- Quality: Clear, recognizable animal sounds
- Volume: Normalized across all files

## Sources

You can find free animal sound effects from:
- Freesound.org
- ZapSplat.com
- SoundBible.com

Make sure to use sounds with appropriate licenses for your use case.
